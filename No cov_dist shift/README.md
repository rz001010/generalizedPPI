# PPI estimators under no covariate distribution shift
This folder contain simulation scripts for PPI estimators under no covariate distribution shift.

Simulations in Sections 6.1, B.2.2, B.2.3, and B.2.4 can be replicated using the scripts in this folder.

- Script for Section B.2.2 has code for generating Figures S.1 and S.2.
- Script for Section B.2.3 has code for generating Figures S.3 and S.4
- Script for Sections 6.1 and B.2.4 has code for generating Figures 1, 2, S.5, and S.6.
